# 🎯 Python Guess Game

A simple and interactive Number Guessing Game developed using Python. In this game, the computer randomly generates a number, and the player attempts to guess it within multiple tries. The program provides hints indicating whether the guessed number is too high or too low until the correct answer is found. This project was created to strengthen Python programming fundamentals while improving logical thinking and problem-solving skills.

---

## 📌 Features

- Random number generation.
- Interactive command-line gameplay.
- Provides hints after every incorrect guess.
- Detects the correct answer and congratulates the player.
- Handles invalid user input.
- Beginner-friendly and well-structured Python code.

## 💡 How It Works

1. Run the Python program.
2. The computer randomly selects a number within a specified range.
3. Enter your guess through the command line.
4. If the guess is incorrect, the program provides a hint indicating whether the number is higher or lower.
5. Continue guessing until the correct number is found.
6. The game congratulates the player after a successful guess.
